# Paw

Portable local AI assistant, compatible with OpenClaw data format.

## Setup

```bash
npm install
npm start
```

## Concept

Two directories, one Claw:
- **Data Directory** — config.json, skills/, memory/, SOUL.md (compatible with `~/.openclaw/`)
- **Workspace Directory** — AGENTS.md, NOW.md, projects (compatible with `~/clawd/`)

Switch machines by pointing to the same directories.
