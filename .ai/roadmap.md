# Roadmap — OpenClaw Lite

## M1 — 骨架可用 ✅
- Electron 壳 + 暗色主题
- Create New / Open Existing 引导
- 单文件夹模式
- config.json 读写
- System prompt 从目录构建
- 单轮/多轮对话（Anthropic + OpenAI）

## M2 — 工具与设置
- Settings 面板（UI 编辑 provider/apiKey/model）
- Streaming 回复（逐字输出）
- 搜索工具（Tavily）
- 代码执行工具（本地 eval）
- 文件读写工具（Claw 目录内）

## M3 — 体验打磨
- 对话历史持久化（保存到 Claw 目录）
- 对话导出 markdown
- Cmd+K 快捷键面板
- 窗口位置/大小记忆
- 代码高亮
- hiddenInset 标题栏恢复

## M4 — 生态兼容
- 兼容 OpenClaw 目录结构（openclaw.json）
- Skill 加载与执行
- Memory graph 集成
- 多会话支持
- 打包分发（.dmg）
