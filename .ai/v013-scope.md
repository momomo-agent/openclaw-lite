# Paw v0.13 — 本地感知

## 范围（三个功能）

### 0. 拖拽增强
- 文本文件（.txt/.md/.js/.ts/.py/.json 等）→ 读取内容注入 prompt
- 图片（.png/.jpg/.gif/.webp）→ vision API 直接分析
- 文件夹 → 生成目录树 + 关键文件摘要
- 拖拽时显示文件类型、大小（不只是文件名 chip）

### 1. 文件预览（新窗口）
- Markdown → Electron 新窗口渲染预览
- 图片 → 新窗口查看（支持缩放）
- 视频/音频 → 新窗口播放
- Web 内容 → 系统默认浏览器打开
- LLM 工具调用产出的文件也能预览
- 预览入口：消息里的文件链接 / 工具调用结果

### 2. 屏幕感知
- 自动：切换到 Paw 窗口时，截取前一个前台窗口
- 手动：用户在对话里表达意图（"看看我屏幕"），LLM 自己识别调截屏工具
- 截图作为图片附件注入对话上下文
- 新增工具：screen_capture（截取屏幕/指定窗口）

## 不做
- 不做 inline 预览（消息内渲染 HTML/webview）
- 不做右侧面板
- 不做 diff 视图
- 不做截屏 UI 按钮（LLM 自己识别意图）
