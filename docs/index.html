What's inside

One Folder = One Workspace
Config, skills, memory, sessions — all in one directory. Copy it anywhere and keep working.
Multi-Agent Chat
Create agents with custom personalities. @mention routing, avatars, group conversations.
Coding Agents
Claude Code, Codex, Gemini CLI as chat participants. Auto-detected, first-class streaming.
20+ Built-in Tools
Search, code exec, file I/O, shell, MCP, cron, web fetch — autonomous multi-round loop.
File Preview
Markdown rendering, syntax highlighting for 50+ languages, Mermaid diagrams, native Quick Look.
Skills & MCP
SKILL.md injection + script execution. Native MCP client with dynamic tool registration.

Shared memory/ directory with real-time file watching and full-text search index.

Anthropic + OpenAI compatible. API key rotation on rate limits. Multiple keys supported.
5 Themes
Dark, Light, Claude, Codex. Per-card status, frosted glass headers, AI-native menubar.
